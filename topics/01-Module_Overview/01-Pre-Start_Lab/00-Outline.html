Pre-Start Lab

Typically the Monday lab sessions will based on the material covered in the preceding Friday's lecture, and will usually consist of an on-line slack session. However, for the first week we will run a more hands-on lab session to help address any issues that people might have with installing the software or interacting with the course material.

To achieve this, the aims of this lab session are:

  • Install / Verify a Suitable Python Environment
    We will be using Python 3.8 distribution and all additional modules that will be used during this course have been tested to ensure no incompatibilities at this time.

  • Review of Basic Data Exploration Skills.
    We (or more accurately you) will take a small dataset based on records of server tips, and explore it using the software tool of your choice. Later we will preform the same analysis using pandas, but for now the focus is on see what you can see when you look at data.

  • Develop Familiarity with Moodle, in Particular Moodle Activities (Quiz)
    To help direct your data exploration, mentioned above, we have a small Moodle quiz based on the server tips. This quiz is formative (i.e., not summative) so does not contribute towards your overall grade. However, it is a good opportunity gain familiarity with Moodle and Moodle quizzes & assignments.
