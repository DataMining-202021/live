Step 2 — Download and import dataset

Download the dataset, tips and place into you working folder. So following the above structure, place it into practicals/01-tips.

Step 3 — Run up a jupyter-lab interface

On Windows (assuming you have anaconda/miniconda) you:

  1. Go to the Windows start menu and select Anaconda Prompt under Anaconda3 (see image).
  2. At the prompt enter jupyter-lab
  3. Your browser should open showing interface similar to than shown below

On linux/OS X you:

  1. Open any terminal
  2. At the prompt enter jupyter-lab and browser should open showing jupyter-lab interface.
