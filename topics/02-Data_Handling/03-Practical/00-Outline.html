Practical for Week 2 (Data Infrastructures)

Data Storage

There is no practical assigned to this topic. Students are welcome to download and configure one or more DBMS with which they are unfamiliar. Most come with example data and queries to test the installation and/or to learn how that DBMS works.

Having installed that DBMS, students are advised to use pandas to read data from an example database managed by that DBMS.
